Table of commands

avatar


# Syntax Description
1 !avatar Returns your avatar.
1 !avatar <user.mention/.id/.name/.nick> Returns the avatar from <user.mention/.id/.name/.nick>.
Examples
# Input Reply
1 !avatar Avatar image
2 !avatar @Deimos Avatar image
3 !avatar Deimos Avatar image

avatar.channel


# Syntax Description
1 !avatar.channel Returns the current channel that logs all avatar changes.
2 !avatar.channel <channel.mention/.id> Define the channel to log all avatar changes in <channel.mention/.id>.

Examples
# Input Reply
1 !avatar.channel No channel to log avatar changes has been assigned yet.
2 !avatar.channel #log-avatar Avatar changes will now be logged in #log-avatar.

ban

Required General Permission: Ban Members


# Syntax Description
1 !ban <user.mention/.id> [--<reason>] Bans <user.mention/.id> from the server
Examples
# Input Action
1 !ban @Deimos Bans @Deimos from the server.
2 !ban 207856807677263874 Bans the user with the ID 207856807677263874 from the server.
3 !ban @Deimos --spamming cute cat gifs Bans @Deimos from the server. Reason: spamming cute cat gifs

clean

Required Text Permission: Manage Messages


# Syntax Description
1 !clean Deletes the last 100 messages that are eithers @Bismarck's messages or commands.
Examples
# Input Reply
1 !clean Deleted 20 messages.

command_level


# Syntax Description
1 !command_level Shows the required Permission Level for all available commands.
Examples
# Input Reply
1 !command_level

command_level.change


# Syntax Description
1 !command_level.change <INTEGER> <command_name> Change the Permission Level for a Command (<INTEGER> min. 0, max. 100).

Examples
# Input Reply
1 !command_level.change 50 ban The command level of ban has been set to 50.

faq


# Syntax Description
1 !faq Lists all available terms.
2 !faq <term> Tries matching <term> and returns its content.
Examples
# Input Reply
1 !faq FAQ list: ping

faq.delete


# Syntax Description
1 !faq.delete <term> Deletes an existing FAQ entry and logs its action in the assigned channel for moderation actions.
Examples
# Input Reply
1 !faq.delete ping Faq ping has been deleted.

faq.edit


# Syntax Description
1 !faq.edit <term> <new_content> Edits an existing FAQ entry and logs its action in the assigned channel for moderation actions.
Examples
# Input Reply
1 !faq.edit ping Pong! Faq ping has been edited, new content: Pong!

faq.new


# Syntax Description
1 !faq.new <term> <content> Creates a new FAQ entry and logs its action in the assigned channel for moderation actions.
Examples
# Input Reply
1 !faq.new ping Pong! New faq ping has been created.
